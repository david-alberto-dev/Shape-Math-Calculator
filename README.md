# 📐 ShapeMath Calculator
A professional, modular Command Line Interface (CLI) application built in Python for arithmetic operations and geometric calculations.

## 📖 Description
ShapeMath was born from the unification of two separate miniprojects: **Simple Calculator** and **Geometry Tool**. This application combines basic arithmetic and geometric calculations, in this case, the area of a 2D shape and the volume of a 3D one, into a single program.

It provides a clean terminal interface where users can perform standard math operations or calculate the area and perimeter of various shapes using an efficient, object-oriented approach.

This project serves as a foundation for my journey into **QA Automation** and **Clean Code** architecture.
